# YouTube MP3 Downloader

โหลดเพลงจาก YouTube เป็น MP3 ผ่านหน้าเว็บ

## วิธีใช้งาน

### Mac
```bash
bash setup-mac.sh
npm start
```

### Windows
```powershell
powershell -ExecutionPolicy Bypass -File setup-windows.ps1
npm start
```

เปิด browser ที่ `http://localhost:3456`

## ไฟล์ MP3 บันทึกไปที่
- Mac: `~/Music/YouTube/`
- Windows: `%USERPROFILE%\Music\YouTube\`
